Project Name: All About Yoga

Name: Olivia Marsolais

Class: Project 1 - Website Development

Purpose: I created this 4-page website to show why yoga is beneficial for health and stress. As an Elementary Education major, I'm interested in how yoga helps people stay balanced. I am a beginner and used this project to learn more about the poses.

Website Features: * Structure: Used semantic HTML tags like header, nav, main, section, and footer.

Pages: Four linked pages (Home, Poses, About Me, and Resources).

CSS Styling: Changed colors to a green theme, updated fonts, and added spacing/padding to make it look clean.

Flexbox: Used Flexbox in the CSS to layout the navigation menu links.

Table: Included a table on the Poses page to list different moves and their benefits.

Multimedia: Embedded a YouTube video at the bottom of the Poses page for a guided workout.

Responsiveness: Added a media query so the navigation menu stacks correctly on a phone screen.
